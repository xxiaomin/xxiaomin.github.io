终端创建后端项目

发表于2023-06-03|更新于2023-06-12|学习笔记
|阅读量:

在vscode,HBuilderX等工具—-在终端用命令行创建后端项目

全局安装(执行一次)

  • 任何一个cmd终端位置安装

局部安装

  • cd到你的项目里，局部安装生成server文件(会与node_modules文件同级)

启动步骤

  • cd到server目录
  • 安装依赖npm install
  • 启动npm start

浏览器输入http://localhost:3000/

  • 一般是3000端口(bin/www里的var port = normalizePort(process.env.PORT || ‘3000’);)
  • 出现以下提示就对了 以上是我学习笔记~~~
